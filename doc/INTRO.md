This is a project of a desktop app to read large (> 1GB) xml files.

## Features

- Read xml files
- Search for specific tags
- Resolve xpath

## Tech Stack

- Tauri
- SvelteKit
- Rust
- TypeScript
- Tailwind CSS
- Quick XML

## Installation

```bash
# Clone the repository
git clone https://github.com/danielvie/xml-reader.git

# Navigate to the project directory
cd xml-reader

# Install dependencies
npm install

# Run the app
npm run dev
```

## Usage

```bash
# Search for a specific tag
npm run search-tag

# Resolve xpath
npm run resolve-xpath

# Get first child
npm run get-first-child
```
